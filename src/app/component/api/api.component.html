Purpose of the API Tab

The API tab compares the current state of the Voice API to its state when this application was created (Summer 2025). Properties are categorized as follows:

check_box The property existed in Summer 2025 and still exists.
close The property existed in Summer 2025 but no longer exists.
help The property was added after 2025.
Property {{ apiProperty.property }} Result @if(apiProperty.type === 'expectedAndFound') { check_box } @else if (apiProperty.type === 'notExpected') { help } @else if (apiProperty.type === 'expectedAndNotFound') { close }
